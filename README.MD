# Select text nodes

Simply select your content, run the plugin, and it will filter out everything but the text nodes, so you can now bulk edit all the text, like change size, font, ....

Works both in Figma and Figjam

## development

1. Clone the project
2. run `npm install`
3. run `npm run watch`
4. In figma/figma, right click -> Plugins -> Development -> import plugin from manifest
5. select the manifest.json file in the source code.
